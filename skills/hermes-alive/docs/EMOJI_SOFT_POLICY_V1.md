# Emoji Soft Policy V1

Marker: `EMOJI_SOFT_POLICY_V1`

Scope: `hermes-alive-v2-dev` only.

Emoji is not a globally forbidden output feature.

Rules:

- Use emoji only when it fits the sentence and relationship context.
- Avoid repeated or decorative emoji stacking.
- Debug, research, high-pressure and serious contexts may naturally use fewer emoji, but code must not forcibly strip them.
- Interruption Policy still enforces `max_bubbles`, content-sharing boundaries and silence decisions.
