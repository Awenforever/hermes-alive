# Phase H matrix and stress suites V1

Markers: `HERMES_ALIVE_MATRIX_SUITE_V1`, `HERMES_ALIVE_STRESS_SUITE_V1`.

The matrix suite verifies Provider readiness, managed configuration, interruption
policy, rich delivery, metadata, interest learning, lifecycle rollback and clean
uninstall behavior.

The stress suite executes the contractual full load by default, reports elapsed
time, file descriptors, thread counts, peak traced memory, bounded files and
failure recoveries. Reduced scale is developer smoke only and is not valid for
final acceptance.
