# Emoji Contextual Policy V2

Marker: `EMOJI_CONTEXTUAL_POLICY_V2`

Scope: `hermes-alive-v2-dev` only.

Rules:

- Emoji is not subject to a global numeric hard cap.
- Use it naturally when it fits the sentence and relationship context.
- Avoid decorative stacking or emoji that distracts from the message.
- Debug, production-operation, audit and serious contexts may naturally use fewer emoji, but emoji is not forbidden.
- Existing code does not strip user/model-provided emoji.
